# proyectofinaltecnicasdeprogramaci-n
**Análisis de Gastos en Salud**

Aplicación gráfica en Python para registrar, consultar, filtrar y visualizar estadísticas sobre gastos en salud personales.

**Descripción del proyecto**

Este proyecto es una herramienta desarrollada en Python con Tkinter que permite llevar un control organizado de los gastos relacionados con la salud.
Incluye funciones para:

**Añadir Registro **:Permite registrar información como tipo de gasto, monto, fecha, categoría, etc.

**Consultar registros **:Muestra todos los datos guardados en pantalla de manera organizada.

**Filtrar gastos por categoría**:Permite ver únicamente los gastos seleccionados

**Visualizar estadísticas**:Genera gráficas que ayudan a analizar la información que desea 

Este proyecto fue creado como trabajo final para la asignatura Técnicas de Programación, demostrando uso de programación modular, manejo de archivos, interfaz gráfica, filtrado de datos y visualización estadística.


**Herramientas utilizadas**
Python 3.13.7        

Tkinter → Para la interfaz gráfica

Pillow (PIL) → Para imágenes

Matplotlib → Para graficación

Pandas → Para manejo de datos
